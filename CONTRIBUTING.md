# 贡献指南

感谢您想为这个 Awesome MCP Servers 列表做出贡献！

## 如何添加新的 MCP 服务器

1. **Fork 本仓库**
2. 编辑 `README.md`，在适当的分类下添加您的条目
3. 条目格式：
   ```markdown
   | **[server-name](https://github.com/org/server)** | 简短描述（10-20字） | — | 维护方 |
   ```
4. 提交 PR

## 条目要求

- ✅ 服务器必须是公开可用且正常维护的
- ✅ 提供准确的描述（中英文均可）
- ✅ 注明维护方（官方 / 社区 / 个人）
- ✅ 尽量标注星标数和客户端兼容性

## 分类说明

请将您的条目放入最合适的分类中。如果现有分类都不合适，可以在 `📋 目录` 后添加新的分类并更新目录链接。
